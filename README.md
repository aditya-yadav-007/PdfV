# PdfV

**PdfV** is a simple and fast **PDF Viewer** for Windows.  
No installation required. Just download and run.

## Highlights
- Clean and minimal interface
- Smooth page navigation
- Zoom in / Zoom out support
- Works on low-end systems
- No ads, no internet required

## How to Use
1. Download the `PdfV.exe` file.
2. Double-click to open it.
3. Click **Open PDF** and choose any `.pdf` file.
4. Scroll to move between pages or use the navigation buttons.
5. Use the slider to adjust zoom level.

## Controls

| Action | How to Do It |
|-------|--------------|
| Open File | Button on top |
| Next Page | Scroll Down / Page Down |
| Previous Page | Scroll Up / Page Up |
| Zoom | Use the Zoom Slider |
| Fit to Screen | Reset Zoom Slider |

## System Requirements
- Windows 7 / 8 / 10 / 11 (64-bit recommended)
- At least 2 GB RAM

## Notes
- The `.exe` is standalone. You **do not** need Python or any external library.
- If Windows SmartScreen shows a warning, choose **Run Anyway** (common for new unsigned apps).
